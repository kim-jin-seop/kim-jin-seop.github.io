# 변수와 상수 선언

## 변수(variable)
변수는 항상 변할 수 있는 값  
선언하는 방법

```
var hello = "hello" 
var world : String = "world!"

print(type(of:a)) // String
print(type(of:b)) // String
```

## 상수(constant)
상수는 변할 수 없는 값. 
선언하는 방법  
```
let hello = 1
let hello : int = 2
```
